## Описание проекта

«Ассистент преподавателя» — настольное приложение для автоматизированной проверки домашних заданий по программированию. Преподаватель создаёт задания с тестовыми данными, загружает файлы решений студентов, запускает проверку и сохраняет результаты в таблицу с возможностью экспорта в CSV.

## Возможности приложения

- Создание и хранение заданий (Java, Haskell)
- Задание тестовых входных данных и ожидаемого вывода
- Автоматическая проверка файлов решений студентов
- Статусы проверки: успешно, неверный ответ, ошибка компиляции, ошибка выполнения, превышено время, среда не найдена
- Таблица результатов с экспортом в CSV
- Локальное хранение заданий и результатов
- Демонстрационный режим для защиты ВКР
- Демонстрационные модули «Импорт из почты» и «Экспорт в Google Sheets»

## Используемые технологии

- Java 17
- JavaFX 21
- Maven
- Gson (хранение заданий и настроек в JSON)

## Запуск приложения

### Требования

1. Установите **JDK 17** (с поддержкой `javac` и `java` в PATH)
2. Установите **Apache Maven**

### Запуск

1. Откройте проект в IntelliJ IDEA, VS Code или Cursor
2. Выполните команду в корне проекта:

```bash
mvn clean javafx:run
```

Приложение откроется с главным окном «Ассистент преподавателя».

## Демонстрационный режим

В разделе **«Настройки»** можно включить **демонстрационный режим**. В этом режиме проверка симулируется без реальной компиляции и запуска кода. Режим отображается заметным баннером в верхней части окна и в разделе «Проверка».

Демо-режим рекомендуется использовать на защите ВКР, если среда Java или Haskell настроена не полностью.

### Пример проверки (Java)

1. Создайте задание «Сумма двух чисел» (Java), вход: `2 3`, вывод: `5`, балл: 10
2. В разделе «Проверка» выберите задание, укажите ФИО и группу
3. Выберите файл `examples/Sum.java`
4. Нажмите «Проверить»

## Демонстрационные модули

Кнопки **«Импорт из почты»** и **«Экспорт в Google Sheets»** в разделе «Настройки» реализованы как расширяемые демонстрационные заглушки. При нажатии отображается сообщение о возможности подключения модулей позже. Пароли, токены и учётные данные в приложении не хранятся.
